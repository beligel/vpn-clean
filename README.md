# VPN Dashboard

Clean VPN dashboard with Hysteria 2, VLESS+Reality, BIND9 DNS, and traffic monitoring.

## Features

- 🚀 Hysteria 2 (UDP port 8443) - systemd managed
- 🛡️ VLESS + Reality (TCP port 443) 
- 🌍 BIND9 DNS Server
- 📊 Traffic History Chart (hourly data collection)
- 🔒 SSL Certificate management
- 🌐 Network stats

## Dashboard Sections

1. 💻 System (CPU, RAM, Disk)
2. 🚀 Hysteria 2 (status, connections)
3. 🛡️ VLESS + Reality (status, connections)
4. 🔒 SSL Certificates
5. 🌐 Network Traffic
6. 🌍 DNS Server (BIND9)
7. 📊 Traffic History (24h chart at bottom)
